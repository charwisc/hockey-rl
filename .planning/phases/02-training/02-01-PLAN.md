       ```python
       def _default_opponent_fn(self, obs: np.ndarray) -> np.ndarray:
           path = self._opponent_path
           if path is None or path == "random":
               return np.random.uniform(-1.0, 1.0, size=(4,)).astype(np.float32)
           if path != self._opponent_model_cache["path"]:
               from stable_baselines3 import PPO

               self._opponent_model_cache["path"] = path
           action, _ = self._opponent_model_cache["model"].predict(obs, deterministic=True)
           return action
       ```

    **training/self_play_callback.py — SelfPlayPoolCallback (per D-01, D-02, D-03, D-04, TRAIN-01):**

    Import `os, random` and `deque` from collections.

    Class `SelfPlayPoolCallback` inheriting from `BaseCallback` (from `stable_baselines3.common.callbacks`).

    Constructor parameters:
    - `pool_dir: str` — directory for pool checkpoint files
    - `pool_update_freq: int = 500_000` — fire every N total timesteps (per D-02)
    - `max_pool_size: int = 20` — evict oldest when exceeded (per D-03)
    - `verbose: int = 0`

    Instance variables:
    - `self.pool_dir = pool_dir`
    - `self.pool_update_freq = pool_update_freq`
    - `self.max_pool_size = max_pool_size`
    - `self._pool: deque[str] = deque()` — ordered list of pool checkpoint paths
    - `self._latest_path: str | None = None`

    In `__init__`, call `os.makedirs(pool_dir, exist_ok=True)`.

    `_on_step(self) -> bool`:
    - Check: `if self.num_timesteps % self.pool_update_freq < self.training_env.num_envs:`
    - If true, call `self._snapshot_and_reassign()`
    - Return True always

    `_snapshot_and_reassign(self)`:
    1. `step = self.num_timesteps`
    2. `path = os.path.join(self.pool_dir, f"pool_step_{step}.zip")`
    3. If `os.path.exists(path)`: skip save (handles resume collision per Pitfall 7), but still reassign
    4. Else: `self.model.save(path)`
    5. `self._pool.append(path)` and `self._latest_path = path`
    6. Eviction loop: `while len(self._pool) > self.max_pool_size:` pop left, `os.remove(old)` if exists
    7. Reassign opponents: for each env index `i` in `range(self.training_env.num_envs)`:
       - If `random.random() < 0.5 or len(self._pool) <= 1`: use `self._latest_path`
       - Else: `random.choice(list(self._pool)[:-1])` (historical only)
       - Call `self.training_env.set_attr("opponent_path", chosen, indices=[i])`
    8. If verbose: `print(f"[SelfPlay] step={step} pool_size={len(self._pool)}")`

    **training/checkpoint_callback.py — WallTimeCheckpointCallback (per D-09, D-10, D-11, D-12, TRAIN-03):**

    Import `time, os`.

    Class `WallTimeCheckpointCallback` inheriting from `BaseCallback`.

    Constructor parameters:
    - `checkpoint_dir: str = "/workspace/checkpoints"` — hardcoded default per D-09
    - `interval_minutes: float = 30.0`
    - `verbose: int = 1`

    Instance variables:
    - `self.checkpoint_dir = checkpoint_dir`
    - `self.interval_seconds = interval_minutes * 60`
    - `self._last_save_time: float = 0.0`

    In `__init__`, call `os.makedirs(checkpoint_dir, exist_ok=True)`.

    `_on_training_start(self) -> None`:
    - `self._last_save_time = time.time()`

    `_on_step(self) -> bool`:
    - `now = time.time()`
    - If `now - self._last_save_time >= self.interval_seconds`: call `self._save()`, set `self._last_save_time = now`
    - Return True

    `_save(self)`:
    1. `step = self.num_timesteps`
    2. `model_path = os.path.join(self.checkpoint_dir, f"step_{step}.zip")` — per D-10
    3. `vecnorm_path = os.path.join(self.checkpoint_dir, f"step_{step}_vecnorm.pkl")` — per D-10
    4. `self.model.save(model_path)`
    5. `vec_norm = self.model.get_vec_normalize_env()`
    6. If `vec_norm is not None`: `vec_norm.save(vecnorm_path)` — per D-12
    7. If verbose: `print(f"[Checkpoint] step={step} saved to {model_path}")`

    **training/tb_callback.py — TensorBoardCustomCallback (per D-13, TRAIN-04):**

    Import `numpy as np`.

    Class `TensorBoardCustomCallback` inheriting from `BaseCallback`.

    Constructor: `verbose: int = 0`. Instance variables:
    - `self._episode_goals: list[float] = []` — 1.0 if any goals scored in episode, else 0.0
    - `self._episode_possessions: list[float] = []` — 1.0 if r_possession > 0 at episode end, else 0.0

    `_on_step(self) -> bool`:
    1. `infos = self.locals.get("infos", [])`
    2. `dones = self.locals.get("dones", np.zeros(0, dtype=bool))`
    3. For each `info, done` in `zip(infos, dones)`:
       - If `done` is truthy (episode ended):
         - `score = info.get("score", [0, 0])` — list of [team0_goals, team1_goals]
         - `goals = sum(score)`
         - `self._episode_goals.append(float(goals > 0))`
         - `self._episode_possessions.append(float(info.get("r_possession", 0.0) > 0))`
    4. Return True

    `_on_rollout_end(self) -> None`:
    1. If `self._episode_goals`: `self.logger.record("hockey/goal_rate", float(np.mean(self._episode_goals)))`
    2. If `self._episode_possessions`: `self.logger.record("hockey/puck_possession_rate", float(np.mean(self._episode_possessions)))`
    3. `self._episode_goals.clear()`
    4. `self._episode_possessions.clear()`
